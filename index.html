Tecnologias FRONT-END

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada na
construção de páginas na Web. Documentos HTML podem ser interpretados por
navegadores.

CSS

CSS (Cascading Style Sheet) é usado para estilizar elementos escritos em HTML. O CSS
separa o conteúdo da representação visual do site.

Java Script

JavaScript é uma linguagem de programação que permite a você implementar itens
complexos em páginas web.
